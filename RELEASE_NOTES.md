# Release v1.0.0 — Initial Release

**Release Date:** 2025-11-18

## What's Included
- Initial public release of `metadata_manager`
- Support for custom metadata schemas
- CRUD operations for metadata
- Built-in validation engine
- JSON, SQLite, and PostgreSQL backend support
- CLI tool with commands:
  - `init`
  - `create`
  - `list`
  - `update`
  - `delete`
- Example schema and sample metadata included
- Updated README with usage and structure

## Known Issues & Future Work
- Additional backends (NoSQL, cloud storage)
- More detailed validation types
- Expand test coverage
- PyPI packaging in future release

## Upgrade Notes
- No previous versions — this is the first release.
